# Committees
## • [[Committee on Moderation and Administration]]
Oversights server events and player behavior, both in Discord and Minecraft.
## • [[Committee on Server Development]]
Develops plugins in need and troubleshoots technical server errors.
## • [[Committee on Builds and Design]]
Builds functional structures for gatherings, events, and aesthetics.
## • [[Committee on Media and Communications]]
Covers and disseminates news about server changes, events, and player lore.
