# Insights
- [[Odysseas]] created his folders in the order:
	**1 - Rough Notes**
		Stores quick notes, anything temporary.
		These quick notes are later revisited to be expanded and nurtured to their relevant folder if necessary.
	**2 - Source Material**
		Stores insights from anything you consume.
		Has subfolders for different sources:
			- Articles
			- Books
			- Videos
	**3 - Tags**
		Stores the tags.
		*Tags come in handy in navigating your vault.* 
		His tags are just empty notes instead of the usual hashtag.
			To create a new tag in his style,  create a new inline note `[[example]]` on the first line of the note you are editing. Enter inside the note and move it to the the Tags folder via the three dot icon on the top right, then "Move file to..." Adding the tag to a different note is the same as referencing it `[[example]]`.
	**4 - Indexes**
		Basically tags that have been referenced many times. If such happens to a tag, it is moved to the Indexes folder. We can now start filling in subheadings to the empty tag note.
			**Example**: a tag named `history` has been referenced many times. Inside the `history` tag, which is an empty note, we can start organizing the links into relevant subheadings:
				`history`
				Roman Empire
					- A link to the note “The Fall of Rome” where `history` is a tag.
					- A link to “Julius Gaius Caesar” where `history` is a tag
				World War II
					- A link to the note “Nazi Party” where `history` is a tag.
					- A link to the note “Imperial Japan” where `history` is a tag.
		You are essentially linking back to the notes that have linked to the tag note, called bidirectional linking.
	**5 - Templates**
		Sets up the templates for your main notes.
	**6 - Main Notes** 
		This is where everything goes. Some people call it the **[[Zettelkasten]]**, which I think I might.
- The lack of folders is what keeps Zettelkasten **flexible** and **open to connection**.
- As your Zettelkasten expands, you’ll naturally begin to see different areas of interest form because of the links; **chaos slowly turns into order**.
- He set the creation location of his notes into the Main Notes folder.
- Explained how [[normal|templates]] are made.

# Note-Taking
- After reading a book, Odysseas would create a new note in the Source Material folder under the title of the book.
- He would format his insights like this:
	- Book title;
	- Page number;
	- A good quotation;
	- Self interpretation of the quotation.
- He then creates a new note in the Main Notes folder to create an entire critique of the book. *The notes from the source material were in context, but these ones, the main notes, are not. It gives you a chance to talk about **the bigger picture**.*

# Effective Tagging
1. Avoid being too ‘vague.’
	- A tag called `self-improvement` could be split into `fitness`, `health`, `productivity`, etc.
2. Avoid being too specific either.
	- You’ll otherwise never be able to use the tag again.
3. Keep tags related to your interests.
	- You could probably use `music`, `editing`, `maths`, etc.
