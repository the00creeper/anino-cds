## About Quartz
[Quartz](https://quartz.jzhao.xyz/) is a static site generator that transforms [markdown](https://en.wikipedia.org/wiki/Markdown) files created with [Obsidian](https://obsidian.md/) to fully functional websites. It uses [GitHub](https://github.com/) to store its files as repositories, which then goes to [GitHub Pages](https://docs.github.com/en/pages) to host the repositories by giving them live website links. 

From markdown to website, the process looks like:
1.  Obsidian: raw markdown (text formatting)
2.  GitHub: repository (file storage)
3.  GitHub Pages: hosting (web link provider)
4. Quartz: static site generator (`.md` to HTML)
