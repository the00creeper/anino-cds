# **ZynxIPChecker v1.1a** Update!
## Action Taken / What Was Added:
- Updated command `/ipinfo <player>` – shows all IP addresses a player has used and lists other accounts that also used those IPs.
- Added command `/accountlookup <ip/player>` – shows all accounts that have logged in from a specific IP.
> [!note]
> - Compatible with existing player data, no changes to current files.
> - Professional display in chat with colored title and organized lists.
## Why It Was Necessary / Benefits:
- Helps admins see if multiple accounts use the same IP – useful for moderation and preventing rule violations.
- Makes it easier to manage players and track suspicious activity.
- Provides organized and professional output for admins to read quickly.
