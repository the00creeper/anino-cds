- Configured [Obsidian](https://obsidian.md/)’s [Dataview Serializer](https://developassion.gitbook.io/obsidian-dataview-serializer) plugin for dynamic Dataview table updates with [[DEV-COS-008-20250920#About Quartz|Quartz]].
- Dataview tables offer a way to enable dynamic updates for tables, meaning new files automatically get added to or removed from the table if they meet or fail to meet specific tag requirements, eliminating the need for manual updating.
- The problem with Dataview queries is that they use code blocks, which gets rendered as it is in Quartz:
```dataview
TABLE date as "Date" 
FROM [[tags/Development Report]] and [[Zynn]] 
SORT file.name DESC
```
- Dataview Serializer continuously scans for Dataview queries throughout the entire vault and converts them to raw static markdown tables, which Quartz can then interpret correctly.
