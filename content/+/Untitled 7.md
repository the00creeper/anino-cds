- `//set <block>`: sets the entire selected region to a block
- `//undo`: undoes everything
- `//replace <toBeReplaced> <replacer>`: replaces a block with another
- `//replace <block>`: replaces every existing block except air with specified block
- `//pos 1`: sets position 1
- `//pos 2`: sets position 2
- `//copy`: copies selected region
- `//paste`: pastes selected region
- `//move <moveIncrement>`: moves selected region to where the player is facing
- `//rotate <90/180/270>`: rotates selected region, requires region to be `//copy` first, `//rotate`, then `//paste` to execute
- `//flip`: mirrors a selected region, same execution as `//rotate`
